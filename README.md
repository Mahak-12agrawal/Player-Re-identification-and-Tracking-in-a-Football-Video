# Player Re-Identification and Tracking in Football Video

This project is an **implementation and adaptation** of an existing open-source project
focused on player detection, tracking, and re-identification in football videos.

The system uses:
- YOLOv8 for object detection
- DeepSORT for multi-object tracking
- OpenCV for video processing
